Text Spacing Examples

This paragraph demonstrates line-height. It controls the vertical space between lines, which is crucial for readability. A generous line-height makes text more comfortable to read, while a tight one can make it feel cramped.

This paragraph demonstrates letter-spacing. It adjusts the horizontal space between individual characters. This can be used for stylistic effects, like making a heading feel more open and modern.

This paragraph demonstrates word-spacing. It modifies the horizontal space between words. This is often used for fine-tuning text alignment and visual flow in a layout.
