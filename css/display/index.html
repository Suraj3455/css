Understanding CSS Display Property

Block Display

A block element always starts on a **new line** and takes up the **full width** available. This is a block-level element. Examples include div, p, and h1.

Inline Display

An inline element **does not** start on a new line and only takes up as much width as necessary. This is an inline element. You **cannot** set its width or height. Examples include span, a, and strong.

Inline-Block Display

An inline-block element is similar to an inline element, but you **can** set its width and height. This is an inline-block element. It flows with the surrounding text but can be styled with dimensions.

None Display

The `display: none;` property **hides** an element completely from the page. It's not just invisible; it is removed from the document flow as if it never existed. The space it would have occupied is also gone. This element is hidden. The element is not rendered, and it doesn't take up any space.
