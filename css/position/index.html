Understanding CSS `position` Property

position: static

This is the default position for all elements. Static elements are not affected by `top`, `bottom`, `left`, or `right` properties. They are positioned according to the normal document flow.

Static Box

position: relative

A relatively positioned element is positioned relative to its normal position. You can use `top`, `bottom`, `left`, and `right` to shift it. The space it would have occupied remains in the document flow.

Relative Box

position: absolute

An absolutely positioned element is positioned relative to its **nearest positioned ancestor** (not static). If no such ancestor exists, it's positioned relative to the initial containing block (the `` element). It's taken out of the normal document flow, so other elements will behave as if it isn't there.

Absolute Box

position: fixed

A fixed element is positioned relative to the viewport. This means it always stays in the same place, even if the page is scrolled. It's also taken out of the normal document flow.
