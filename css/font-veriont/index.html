CSS `font-variant` Property

This is a normal paragraph to serve as a baseline for comparison. The text here appears exactly as typed.

`font-variant-caps`

This demonstrates `font-variant: small-caps`. lowercase letters become small capitals, while uppercase letters remain large.

`font-variant-numeric`

Here are some numbers: 1234567890. This `font-variant-numeric: tabular-nums` makes numbers have the same width. Compare that to this: 1234567890.

`font-variant-ligatures`

This text uses `font-variant-ligatures: common-ligatures`. The 'fi' and 'fl' combinations might be rendered as a single character.
