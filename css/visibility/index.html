Understanding CSS `visibility` Property

Visible Element

This is a paragraph of text. Below is a box that is **visible**.

I am a visible box.

This text is placed after the visible box, and it flows normally.

Hidden Element

This is a paragraph of text. Below is a box that is **hidden**.

I am a hidden box.

Notice how this text is placed where the hidden box would have been. The hidden box still takes up space in the document flow.
